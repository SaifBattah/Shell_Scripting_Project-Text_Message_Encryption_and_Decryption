project run steps:
#1. use this command to run project
	$ ./project.txt

#2. for Encryption:
	press 'e' then hit 'Enter'
	Enter plain file name which is : 'plain.txt' then hit 'Enter'
	Enter cipher file name which is : 'cipher.txt' then hit 'Enter'

#3. for Decryption:
	press 'd' then hit 'Enter'
	Enter cipher file name which is : 'cipher.txt' then hit 'Enter'
	Enter plain file name which is : 'plain.txt' then hit 'Enter'
